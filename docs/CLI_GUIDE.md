# คู่มือ CLI Commands

รันผ่านคำสั่ง:
```
php console <command> [arguments]
```

## คำสั่งหลัก
- serve [host] [port]
- migrate
- seed
- cache:clear
- make:controller <name>
- make:model <name>
- make:middleware <name>
- test
- help

## ตัวอย่างการใช้งาน
- php console serve
- php console migrate
- php console seed
- php console make:controller ProductController
- php console test

ดูรายละเอียดการทำงานของแต่ละคำสั่งในไฟล์ console
